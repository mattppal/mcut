# Function: getRenderableElements()

```ts
function getRenderableElements(project, timeMs): RenderableElement[];
```

Defined in: [packages/timeline/src/transitions.ts:132](https://github.com/mattppal/mcut/blob/main/packages/timeline/src/transitions.ts#L132)

Every element that needs pixels at `timeMs` — the active set plus
transition partners outside their own ranges. Render paths (compositor,
export prepare, preview pool) must all enumerate through this so they
agree on what is on screen.
