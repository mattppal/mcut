# Function: applyCommand()

```ts
function applyCommand(project, command): object;
```

Defined in: [packages/timeline/src/commands.ts:128](https://github.com/mattppal/mcut/blob/main/packages/timeline/src/commands.ts#L128)

Validate and apply a command to a project. Pure: returns a new project,
never mutates. Throws [CommandError](../classes/CommandError.mdx) on unknown commands, invalid
payloads, or violated invariants (unknown ids, overlapping elements, ...).
