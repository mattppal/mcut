# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories:

https://github.com/mattppal/mcut/security/advisories/new

Do not open a public issue for a vulnerability until it has been reviewed and a
fix is available.

## Supported versions

mcut is currently in alpha. Security fixes target the latest published alpha
release of each package unless a maintainer explicitly announces broader
support.

## What to include

- Affected package and version.
- Runtime, browser, or OS details where relevant.
- A minimal reproduction or proof of concept.
- Any known impact, workaround, or exploitability notes.

## Disclosure

Maintainers will acknowledge valid reports as soon as practical, coordinate a
fix privately, and publish an advisory when the fix is released.
